Hexo 使用手册

什么是 Hexo？

Hexo 是一个快速、简洁且高效的 Node.js 静态博客框架。它使用 Markdown 解析文章，在几秒内即可生成静态网页。

四、核心命令

命令 简写 说明
hexo init — 初始化站点
hexo new <title> hexo n 新建文章
hexo generate hexo g 生成静态文件
hexo server hexo s 启动本地服务器预览
hexo deploy hexo d 部署到远程服务器
hexo clean — 清除缓存和已生成文件
hexo publish <title> hexo p 发布草稿
hexo list — 列出所有路由

五、预览

  • 默认访问：http://localhost:4000
  • 草稿预览：hexo s --draft
  • 自定义端口：hexo s -p 5000
  • 静态模式（仅 public 目录）：hexo s -s

十二、进阶技巧

1. 自定义永久链接

使用 hexo-abbrlink 插件生成短链，避免中文文件名在 URL 中显示。

2. 资源文件夹

在 _config.yml 中启用 post_asset_folder: true，创建文章时会自动生成同名文件夹存放图片等资源。

3. 多语言支持

将 language 设为 zh-CN（简体中文）、zh-TW（繁体中文）或 en（英文）。

4. Front-matter 中控制文章可见性

十三、常见问题

问题 解决方案
hexo: command not found 检查 Node.js 和 npm 全局安装路径
页面白屏 / 样式丢失 尝试 hexo clean && hexo g
部署后 404 检查仓库名是否正确（应与用户名一致）、分支是否正确
Markdown 中图片不显示 启用 post_asset_folder 并用 {% asset_img %} 标签
主题配置不生效 确认是在站点 _config.yml 还是主题的配置文件中修改
